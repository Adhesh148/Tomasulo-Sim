# Tomasulo Simulation using Python & Verilog
--------------------------------------------

In this program simulate the tomasulo processor scheduling algorithm using Python & Verilog. The simulation code is written in python and ALU modules are coded in verilog.

### How to run
--------------

> Goto src folder

> Run the following code on the terminal:
	python3 tomasulo.py

The instruction input to the algorithm can be changed by modifying the **'instr.txt'** folder in src folder.
